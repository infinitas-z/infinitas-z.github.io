什么是JSP页面

jsp(java server pages): java 服务端页面，一个既可以写java也可以写html的页面

request请求对象

相关概念

  • request请求方式：最常用的是GET请求和POST请求
  • GET请求：通常在超链接中使用
  • GET特点：请求的参数在URL后出现且有长度限制，安全性不高
  • POST请求：通常在表单提交中使用，中文需要编码处理
  • POST特点：请求参数在请求体中，没有长度限制，安全性高
